# chat-gpt-plus

## 技术架构
- 管理平台后端：**springboot 2.7.10**
- 管理平台前端：**element-ui**
- 前台H5：**uni-app**
- 接口API：**Restful**

## uni-app H5端 
### 实现功能：

基于微信公众号开发，已实现：
- [√] 微信登录
- [√] 微信JSAPI支付
- [√] 微信Native支付
- [ ] 微信PC扫码登录
- [ ] 免签支付
	- [ ] 微信免签支付
	- [ ] 支付宝支付，面对面支付等


前端已经支持stream式输出，支持连续对话、国内高速代理，内容秒回复、内置违规词检测、AI绘图和大量预设模型等，同时支持充值购买使用，兑换码，分享，和邀请获取次数

微信登录、聊天对话、模型单独展示、邀请、兑换码、签到、分享、提现、使用帮助、意见反馈、充值（H5支付，扫码支付）、使用统计
